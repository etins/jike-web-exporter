# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-22

### Added
- Initial public release of `jike-export.js`.
- Auto-scroll loading for timeline pages.
- Multi-source extraction strategy:
  - network response capture
  - page state snapshot parsing
  - DOM fallback parsing
- Export to both JSON and CSV in one run.
- Basic project docs and publish checklist:
  - `README.md`
  - `README.zh-CN.md`
  - `PUBLISH_CHECKLIST.md`
  - MIT `LICENSE`
